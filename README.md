\# FlyRank Week 4 - The Polite Scraper



\## Overview

This project demonstrates a polite web scraper built with Python.



The scraper:

\- Fetches data from BooksToScrape

\- Extracts book information

\- Cleans and structures data

\- Saves records to CSV

\- Uses a custom User-Agent

\- Implements rate limiting

\- Respects robots.txt



\## Technologies Used

\- Python

\- Requests

\- BeautifulSoup4

\- Pandas



\## Output

The scraper generates a CSV file containing:

\- Book Title

\- Price

\- Availability



\## Run



Install dependencies:



pip install -r requirements.txt



Run scraper:



python scraper.py
